# WeedioTool v1.0.0

Universal media downloader — videos, audio, and images from any website.

## Downloads

| Platform | File | Install |
|----------|------|---------|
| **Mac** (Intel + Apple Silicon) | `Mac/WeedioTool-1.0.0-universal.dmg` | Drag to Applications |
| **Mac** (ZIP alternative) | `Mac/WeedioTool-1.0.0-universal-mac.zip` | Unzip, move to Applications |
| **Windows** (Installer) | `Windows/WeedioTool-Setup-1.0.0.exe` | Double-click, follow steps |
| **Windows** (Portable) | `Windows/WeedioTool-1.0.0-portable.exe` | Run directly, no install |

## Supported Sites (1800+)

YouTube, Instagram, LinkedIn, Twitter/X*, TikTok, Facebook, Vimeo, Dailymotion, Reddit, and 1800+ more via yt-dlp.

*Twitter/X requires browser login — the app will guide you through the steps.

## Features

- Paste URL > pick quality > download as MP4
- Auto-detects content type (video, image, mixed)
- H.264 MP4 output (plays on any device — Mac, Windows, iPhone, Android)
- Batch download (multiple URLs at once)
- Clip sections from YouTube videos (timestamp-based)
- Image scraping from news/article websites
- Download history with re-download support
- System tray — minimizes and downloads continue in background

## Future-Proof Design

- yt-dlp auto-updates on every launch (fixes site changes within hours)
- No manual updates needed for download compatibility
- ffmpeg bundled — no external dependencies
- SQLite history — lightweight, no server needed
