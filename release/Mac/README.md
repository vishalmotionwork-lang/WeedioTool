# WeedioTool — Mac Installation

## Quick Install (DMG — Recommended)

1. Double-click `WeedioTool-1.0.0-universal.dmg`
2. Drag **WeedioTool** into the **Applications** folder
3. Open WeedioTool from Applications
4. If macOS says "unidentified developer":
   - Go to **System Settings > Privacy & Security**
   - Scroll down and click **"Open Anyway"** next to the WeedioTool message
   - Or: Right-click the app > **Open** > click **Open** in the dialog

## Alternative Install (ZIP)

1. Unzip `WeedioTool-1.0.0-universal-mac.zip`
2. Move `WeedioTool.app` to your Applications folder
3. Open it (same security steps as above if needed)

## Works on both Intel and Apple Silicon Macs (universal binary).

---

## How It Works

- Paste any URL (YouTube, Instagram, LinkedIn, TikTok, etc.) and click **Grab**
- Choose quality/format and click **Download Selected**
- Files save wherever you choose (file picker opens each time)

## Auto-Updates

- **yt-dlp** (the download engine) updates automatically on every launch
- This means if YouTube/Instagram/etc. changes their site, WeedioTool fixes itself
- No manual updates needed for download compatibility

## Troubleshooting

| Problem | Solution |
|---------|----------|
| "App is damaged" | System Settings > Privacy & Security > Open Anyway |
| Slow first launch | yt-dlp is downloading (~37MB one-time) |
| Twitter/X not working | Log in to X in Brave/Chrome first, restart WeedioTool |
| Video won't play | Already fixed — outputs H.264 MP4 (universal playback) |
| App disappeared from dock | Check system tray (top-right menu bar) — click to reopen |
